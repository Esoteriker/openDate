# DateCraft - AI-Powered Date Idea Generator

DateCraft is a modern web application that helps couples generate creative and meaningful date ideas based on mood, budget, available time, and setting.

## About

DateCraft generates a detailed, emotion-driven date plan with step-by-step flow, playlist mood, conversation prompts, and a small surprise idea.

Live website: https://opendate.vercel.app

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build for Production

```bash
npm run build
npm run start
```
